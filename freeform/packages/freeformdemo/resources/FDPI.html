Processing Instructions Demo

Since version 1.1.1 Freeform framework allows you to embed PHP code into the templates. The code is executed every time the document processor reaches it; the code has read only access to the template variables so that it can use them. The code should output valid markup with the help of the echo or print statements; you cannot mix markup and PHP code in a single instruction.
